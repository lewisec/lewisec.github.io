别被“开光”忽悠了

发表于 2014-02-28   |   分类于 慈悲之心   |     |   阅读次数

今年家人本命年，从习俗角度说，最好身上有一些红色的物件。为遂心愿，我在淘宝上开始搜各类编制的红色手绳、手串。

不搜不知道，原来淘宝上卖这类东西的还真不少，很多卖家都说自己的东西经过高僧大德在寺庙里给开过光的，非常辟邪云云。为了让买家信任，有的声称可以出具“开光证书”。我不免揣测，不知道需要通过开光考试，才能拿到这证书？中国人真是有“证书情节”啊。

更有邪乎的，有家专卖各类风水物件的皇冠店，叫慈X阁的，给“开光证书”都觉得不够给力，在宝贝详情页直接贴了一个五分钟的视频，据说是一位大师站在寺院的佛像前，对着摆了一供桌的手串念念有词。末了，卖家还这样写道：“其他卖家的这种假开光实为妄语欺骗，不但不会有任何好的作用，反而因为沾上了妄语的恶因，对您起到坏的作用。”从这段话看，卖家可谓业界良心啊！拳拳之心，催人尿下！

我虽未皈依，还不能称自己是佛弟子，但亲近佛教这么几年，基本常识还是有的。实在不堪忍受这么恶劣的欺骗，不得以，在搜索关键字里补上了：“- 开光”的字样，把宝贝名称带有开光的全部过滤掉，眼不见心静。

开光，这个从小就听了很多遍的词儿，一直给人一种神秘感。小时候我也和大多数人一样，认为有僧人给开光的物品，非常灵验，具有强大的法力。后来亲近了佛教，查了很多资料以后才闹明白，被忽悠了很多年。

在佛教里，开光指的是，在佛像落成之后，如法如律的举行仪式供奉起来。仪式较为繁琐，最后有一环节是用笔象征性的在佛像眼睛上点之。可在佛教辞典中查到。

注意，特指的是给佛教开光，而不是什么手绳、手串，更不是神马日常用品！那种拿个Iphone5去让出家人给开光的行为，我只能呵呵了。
